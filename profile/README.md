# Open Source Kingdom

## Mission
The organization was created from a desire to express Christian values of serving and building others up: that is to say that all repositories and work within the OSK namespace are free to use and will never be licensed under any required fee whatsoever. Tools to build software should be made free to help everyone focus on what matters to them. 

## Overview
As such, OSK is a collection of repositories, applications, frameworks, and other works that are meant to help developers save time and focus on what matters to them rather than all the pieces that are required for any project. OSK is meant to be broadly serving others where it and the community can provide benefit, so while it may seem that a focus of the organization is a specific language or system, it's merely a result of the needs of the community developing it rather than a limitation of the scope of the organization. This organization can and might encompass a variety of languages and systems, so feel free to check back later to see what might have been cooked up next. 

## Community
OSK is meant to be for others, and as such external help and discussion is welcome. Please checkout the [Hub](https://opensourcekingdom.github.io/OSK.Hub/) repository to get a general overview of the code bases and systems currently available and planned for development as well as next steps to getting started. 

We ask all participants to follow our [Code of Conduct](CODE_OF_CONDUCT.md).
